## 📖 Glossário de Termos

### Técnicos

- **Next.js:** Framework React para aplicações web
- **Prisma:** ORM (Object-Relational Mapping) para banco de dados
- **Shadcn/ui:** Biblioteca de componentes UI
- **Recharts:** Biblioteca de gráficos
- **Supabase:** Plataforma de banco de dados PostgreSQL
- **Vercel:** Plataforma de deploy/hospedagem

### Funcionais

- **Dashboard:** Painel principal com visão geral
- **CRUD:** Create, Read, Update, Delete (criar, ler, atualizar, deletar)
- **Budget:** Orçamento mensal
- **Conta Recorrente:** Despesa que se repete mensalmente
- **Transaction:** Registro de uma despesa ou receita
- **Category:** Forma de organizar gastos (ex: Alimentação, Transporte)
- **MVP:** Minimum Viable Product (produto mínimo viável)
